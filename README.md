## 📂 Featured Projects

### 🧠 [SignalNote](https://github.com/MChakrabartyy/SignalNote) — AI Feedback Analysis Agent
*what happens when you stop reading feedback manually and let AI surface the signal*

Built on **Cloudflare Workers, D1, KV, and Workers AI** — SignalNote ingests user feedback and surfaces patterns, sentiment trends, and actionable insights automatically. the kind of thing a PM dreams about having.

`TypeScript` `Cloudflare Workers` `Workers AI` `D1` `KV`

---

### 🤝 [cf_ai_collaborative_debugger](https://github.com/MChakrabartyy/cf_ai_collaborative_debugger) — Real-Time AI Code Debugger
*collaborative debugging, with Llama 3.3 70B as your pair programmer*

Real-time multiplayer debugging powered by **Cloudflare Durable Objects** for state sync and **Llama 3.3 70B** for AI-assisted error analysis. built to explore what edge-native AI architectures can actually do.

`TypeScript` `Cloudflare Workers` `Durable Objects` `Llama 3.3 70B`

---

### 📓 [Monolog](https://github.com/MChakrabartyy/Monolog-webapp) — AI Reflection Journal

`TypeScript` `LLMs` `React`
